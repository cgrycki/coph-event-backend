# College of Public Health Event Application: Backend

### Directory Structure

```
/coph-event-backend/
|-- auth/           API Authentication
|-- bin/            Files to run server, room data
|-- docs/           Documentation generated by jsdoc2markdown
|-- events/         API Events; get, post, patch, delete
|-- rooms/          API Rooms; get rooms, schedules, and availabilities
|-- utils/          Various utils for parameter validation, etc.
|-- README.md       You're reading this file now.
|-- sample.env      A stub environment file with the expected keys for ENV variables.
|-- server.js       Our server tying everything together
```
